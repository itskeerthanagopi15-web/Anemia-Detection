Anemia Detection using Deep Learning

This project is an AI-based Anemia Detection System developed using Python, Deep Learning, Flutter, and TensorFlow Lite. The system predicts anemia symptoms from images such as palm, nail, and eye using a CNN model.

Features
Detects possible anemia symptoms
Uses Deep Learning CNN model
Image-based prediction
Flutter mobile application
Fast prediction using TensorFlow Lite
Simple and user-friendly interface

Technologies Used
Python
TensorFlow
Keras
CNN (Convolutional Neural Network)
Flutter
Dart
HTML/CSS
SQLite

Project Modules
Eye Image Detection
Nail Image Detection
Palm Image Detection
Prediction History
Mobile App Interface

How It Works
User uploads image
Image is processed
CNN model analyzes image
System predicts anemia condition
Result is displayed in app
